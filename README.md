# LangChain-Based-News-Research-System
RAG-based news research tool using LangChain and FAISS vector database
A RAG (Retrieval-Augmented Generation) powered application that:

Scrapes news articles from URLs
Indexes them in a searchable vector database
Answers questions about the articles with source attribution
Provides accurate, context-aware responses based on actual article content (not hallucinations)
